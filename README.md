# Recyclo 🌱♻️

An application designed to facilitate the recycling of various unusable products, aimed at improving waste management and environmental sustainability.

## Features
- Register, track, and schedule pickups of recyclable items.
- Categorize items based on material (plastic, glass, etc.).
- Calculate rewards/points for users based on the amount recycled.

## Tech Stack
- Python (Flask for backend)
- SQLite (database for storing user data)
- HTML, CSS (front-end)

## How to Run
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app using `python app.py`
